# HR Analytics Dashboard – Power BI

## Project Overview
This project analyzes employee attrition trends using Power BI, Excel, and Power Query. The dashboard helps identify workforce patterns, retention issues, and HR performance metrics.

## Tools & Technologies
- Power BI
- Power Query
- Excel
- DAX
- Data Modeling

## Key Insights
- Identified salary disparity as a major factor influencing attrition
- Found high employee exits within the first year
- Analyzed attrition trends across departments and job roles
- Evaluated job satisfaction and tenure-related patterns
- Built interactive dashboards for HR decision-making

## Features
- Interactive Power BI dashboards
- KPI cards and attrition metrics
- Department-wise filtering
- DAX calculations and measures
- Data transformation using Power Query
